## TukMeet Project

> 한국공학대학교 소프트웨어전공 Back-End Project <br>
> 개발기간: 2023-05-05 ~ 진행중

## 프로젝트 소개

본 프로젝트는 Spring Boot를 활용하여 일정 조율을 위한 서비스를 제공하는 웹을 만들기 위한 프로젝트입니다.<br>
이 서비스는 모임, 회의, 파티, 팀 회의 등을 계획할 때 각 참가자의 일정을 고려하여 가장 적합한 일정을 찾아주기 위한 서비스입니다.
